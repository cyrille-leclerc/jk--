Name of a Docker image (with optional tag) which will respond to various commands, specified as an environment variable COMMAND. Should declare a VOLUME at /ws where a workspace will be mounted, for commands which need one (but should check for an environment variable WORKSPACE possibly giving an alternate location); should declare an ENTRYPOINT which be passed a command as its first argument, according to usage. Certain commands expect output on stdout; informational messages should be sent to stderr.
