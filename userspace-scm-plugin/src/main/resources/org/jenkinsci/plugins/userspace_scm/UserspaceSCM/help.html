An SCM implemented using a Docker image of your choice. Environment variables passed:

HEAD
A branch name, tag name, etc. Format up to the implementation.
REV
An exact revision, if not floating according to HEAD. Format up to the implementation.

Commands to be implemented:

checkout
Check out (clone, update, etc.) sources to the workspace.
identify
Return a revision string for the current workspace.
compare
Compare an earlier revision, given as BASELINE, to the current head. A workspace will be mounted if one was requested for polling. Return a change token followed by a space followed by a new revision, where the token may be one of:
NONE
No change: identical.
INSIGNIFICANT
Different, but only in ignorable ways.
SIGNIFICANT
Different.
INCOMPARABLE
Hard to even compare.
