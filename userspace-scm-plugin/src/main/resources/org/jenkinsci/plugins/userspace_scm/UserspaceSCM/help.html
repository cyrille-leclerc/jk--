An SCM implemented using a Docker image of your choice. Commands to be implemented:

checkout

Check out (clone, update, etc.) sources to the workspace. Variables:

HEAD
A branch name, tag name, etc. Format up to the implementation.
REVISION
An exact revision, if not floating according to HEAD. Format up to the implementation.
BASELINE
A previous revision, if available.
CHANGELOG
true or false according to whether a changelog was requested.

identify

Print a revision string for the current workspace.

compare

Compare an earlier revision to the current head. A workspace will be mounted if one was requested for polling. Print a change token followed by a newline followed by a new revision, where the token may be one of:

NONE
No change: identical.
INSIGNIFICANT
Different, but only in ignorable ways.
SIGNIFICANT
Different.
INCOMPARABLE
Hard to even compare.
